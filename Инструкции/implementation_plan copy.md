# Шаг 3: Мастерская креативов (Генерация картинок)

В этом плане описывается техническая реализация 3-го шага, где сгенерированные текстовые сценарии превращаются в готовые картинки (png/jpeg) прямо в браузере без использования платных API.

## Proposed Changes

Мы добавим новую страницу "Мастерская" и интегрируем библиотеку для рендера картинок.

### 1. Установка зависимостей
#### [NEW] `npm install html-to-image`
Библиотека для конвертации HTML/React компонентов в картинки (PNG).
#### [NEW] `npm install lucide-react` (дополнительные иконки)

---

### 2. Подключение на странице Сценариев
#### [MODIFY] `src/app/project/[id]/scripts/page.tsx`
- В карточку каждого сгенерированного скрипта рядом с кнопкой "Копировать таблицу" добавится новая яркая кнопка **"🎨 В Мастерскую"**.
- Кнопка будет вести на URL `/project/[id]/studio/[scriptId]`.

---

### 3. Создание "Мастерской"
#### [NEW] `src/app/project/[id]/studio/[scriptId]/page.tsx`
Создание абсолютно новой страницы, которая будет разделена на 2 экрана:

**Левая панель (Настройки):**
- **Выбор концепции:** Переключение между 3 сгенерированными концепциями (Текст автоматически подтягивается).
- **Редактор текста:** Поля ввода для "Заголовка (Хук)", "Основного текста", "Текста на кнопке".
- **Дизайн:** Выбор шаблона (Минимализм, Градиент, Мем).
- **Цвета:** Пипетки для настройки фона, текста и акцентов (автоматически подтягиваются из промпта).
- **Фон:** Кнопка для загрузки фоновой картинки (если это мем или нужно фото продукта).

**Правая панель (Превью и Скачивание):**
- Интерактивный холст (Canvas), который в реальном времени отображает изменения текста, цветов и фона.
- Выглядит как реальный рекламный креатив (например, с красивой кнопкой внизу).
- Большая кнопка **"📥 Скачать PNG"**. При нажатии `html-to-image` делает "скриншот" этого холста в высоком качестве и сохраняет файл на компьютер пользователя.
