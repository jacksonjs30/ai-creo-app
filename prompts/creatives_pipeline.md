## ЛОГИКА ФОРМАТОВ

### ЯКЩО ФОРМАТ = КАРТИНКА (Статика)
**Генеруємо:**
1. Тексти: Хук, Біль, Рішення, CTA.
2. ПОВНЕ ТЗ ДЛЯ ДИЗАЙНЕРА (Бренд-гайдлайн, палітра, розташування елементів, розмір, референс).
**Правило:** Текст на картинці — не більше 13–22 слів.

### ЯКЩО ФОРМАТ = ВІДЕО
**Генеруємо:**
1. Ідея і Хук.
2. ПОВНИЙ СЦЕНАРІЙ ДЛЯ МОНТАЖЕРА (Таймінг, Відеоряд, VO — Voice Over, TBE — Text By Eye, Музика/Звук).
**Правило:** Сценарій розписаний по секундах (напр. 0-3 сек).

## ЗОЛОТІ ПРАВИЛА (ОБОВ'ЯЗКОВО)
- **ЕМОЦІОНАЛЬНІСТЬ:** Пиши про почуття, не факти.
- **СПЕЦИФІЧНІСТЬ:** Конкретні цифри, час, суми.
- **ПОРТРЕТНІСТЬ:** Для конкретної людини, не для всіх.
- **КОНТРАСТ:** Було ПЕКЛО vs стало ДОБРЕ.
- **УЗНАВАЄМІСТЬ:** Люди впізнають себе в сценаріях.
